# VTC_numerical_experiments

Here's the script for the numerical computation in our submitted paper to VTC 2022: Handover Skipping Analysis in Dense Cellular Network Using Poisson Cluster Process.



### repository file descriptions.

main_cDRHOR.py: the main script.

param_set4_lambda-p-10.csv: the input csv file for the case λp=10 (the 1st argument that should be added on the command line).

param_set4_lambda-p-5.csv  : the input csv file for the case λp=5   (the 1st argument that should be added on the command line).
